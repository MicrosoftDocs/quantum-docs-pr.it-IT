Pacchetto [](https://nuget.org/packages/)


Restituisce true se e solo se due input non sono quasi uguali (ovvero, non sono compresi nella tolleranza di 1e-12).

```qsharp
function NotNearlyEqualD (a : Double, b : Double) : Bool
```
