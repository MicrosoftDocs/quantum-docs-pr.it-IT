# <a name="complexpolarascomplex-function"></a>ComplexPolarAsComplex (funzione)

Spazio dei nomi: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Pacchetto: [Microsoft. Quantum. standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Converte un numero complesso di tipo `ComplexPolar` in un numero complesso di tipo `Complex` .

```qsharp
function ComplexPolarAsComplex (input : Microsoft.Quantum.Math.ComplexPolar) : Microsoft.Quantum.Math.Complex
```


## <a name="input"></a>Input

### <a name="input--complexpolar"></a>input: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Numero complesso $c = r e ^ {i t} $.



## <a name="output--complex"></a>Output: [complesso](xref:Microsoft.Quantum.Math.Complex)

Numero complesso $c = x + i y $.
