# <a name="snd-function"></a>SND (funzione)

Spazio dei nomi: [Microsoft. Quantum. Canon](xref:Microsoft.Quantum.Canon)

Pacchetto: [Microsoft. Quantum. standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Data una coppia, restituisce il secondo elemento.

```qsharp
function Snd<'T, 'U> (pair : ('T, 'U)) : 'U
```


## <a name="input"></a>Input

### <a name="pair--tu"></a>coppia: (t,' U)

Tupla con due elementi.



## <a name="output--u"></a>Output:' U

Secondo elemento di `pair` .

## <a name="type-parameters"></a>Parametri di tipo

### <a name="t"></a>T

Tipo del primo membro della coppia.
### <a name="u"></a>' U

Tipo del secondo membro della coppia.
