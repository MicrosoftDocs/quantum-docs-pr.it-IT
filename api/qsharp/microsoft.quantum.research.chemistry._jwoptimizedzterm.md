# <a name="_jwoptimizedzterm-operation"></a>Operazione _JWOptimizedZTerm

Spazio dei nomi: [Microsoft. Quantum. Research. Chemistry](xref:Microsoft.Quantum.Research.Chemistry)

Pacchetto [](https://nuget.org/packages/)


Applica l'evoluzione del tempo in base a un termine Z descritto da un oggetto `GeneratorIndex` .

```qsharp
operation _JWOptimizedZTerm (term : Microsoft.Quantum.Simulation.GeneratorIndex, stepSize : Double, parityQubit : Qubit, qubits : Qubit[]) : Unit
```

`GeneratorIndex` che rappresenta un termine Z.


### <a name="stepsize--double"></a>stepSize: [Double](xref:microsoft.quantum.lang-ref.double)

Qubit che determina il segno di evoluzione del tempo.

Qubits di hamiltoniana.
