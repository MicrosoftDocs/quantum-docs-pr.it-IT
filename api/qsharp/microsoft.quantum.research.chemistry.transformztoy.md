# <a name="transformztoy-operation"></a>Operazione TransformZToY

Spazio dei nomi: [Microsoft. Quantum. Research. Chemistry](xref:Microsoft.Quantum.Research.Chemistry)

Pacchetto: [Microsoft. Quantum. Research. Chemistry](https://nuget.org/packages/Microsoft.Quantum.Research.Chemistry)


Usato per modificare la base di un operatore Z in un operatore Y per coniugazione.

```qsharp
operation TransformZToY (qubit : Qubit) : Unit is Adj + Ctl
```


## <a name="input"></a>Input

### <a name="qubit--qubit"></a>Qubit: [qubit](xref:microsoft.quantum.lang-ref.qubit)

Qubit di cui è necessario modificare la base.
