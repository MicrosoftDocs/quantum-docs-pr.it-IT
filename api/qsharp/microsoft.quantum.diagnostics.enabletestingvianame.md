# <a name="enabletestingvianame-user-defined-type"></a>Tipo definito dall'utente EnableTestingViaName

Spazio dei nomi: [Microsoft. Quantum. Diagnostics](xref:Microsoft.Quantum.Diagnostics)

Pacchetto: [Microsoft. Quantum. QSharp. Core](https://nuget.org/packages/Microsoft.Quantum.QSharp.Core)


Attributo riconosciuto dal compilatore tramite il quale è possibile definire un nome alternativo che può essere usato durante il caricamento di un tipo o chiamabile a scopo di test.

```qsharp

@ Microsoft.Quantum.Core.Attribute()
newtype EnableTestingViaName = (String);
```
