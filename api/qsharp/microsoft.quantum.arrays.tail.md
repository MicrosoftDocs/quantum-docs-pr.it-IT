# <a name="tail-function"></a>Tail - funzione

Spazio dei nomi: [Microsoft. Quantum. Arrays](xref:Microsoft.Quantum.Arrays)

Pacchetto [](https://nuget.org/packages/)


Restituisce l'ultimo elemento della matrice.

```qsharp
function Tail<'A> (array : 'A[]) : 'A
```


## <a name="input"></a>Input

### <a name="array--a"></a>array:' A []

Matrice di cui viene utilizzato l'ultimo elemento. La matrice deve avere una lunghezza almeno pari a 1.



## <a name="output--a"></a>Output:' A

Ultimo elemento della matrice.

## <a name="type-parameters"></a>Parametri di tipo

### <a name="a"></a>' A

Tipo degli elementi della matrice.
