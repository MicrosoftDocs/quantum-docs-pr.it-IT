# <a name="q-libraries"></a>Librerie Q# #

La libreria standard Q# include operazioni, funzioni e tipi definiti dall'utente usati per sviluppare programmi quantistici. Sono incluse anche tre librerie aggiuntive basate sulla libreria standard, ovvero le librerie di chimica, per il calcolo numerico e di Machine Learning.

Per altre informazioni sulle librerie Q#, vedere <xref:microsoft.quantum.libraries>.
