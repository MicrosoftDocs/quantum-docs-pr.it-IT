# <a name="multim-operation"></a>Operazione multifunzione

Spazio dei nomi: [Microsoft. Quantum. Measurement](xref:Microsoft.Quantum.Measurement)

Pacchetto: [Microsoft. Quantum. standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Misura ogni qubit in una matrice specificata in base allo standard.

```qsharp
operation MultiM (targets : Qubit[]) : Result[]
```


## <a name="input"></a>Input

### <a name="targets--qubit"></a>destinazioni: [qubit](xref:microsoft.quantum.lang-ref.qubit)[]

Matrice di qubits da misurare.



## <a name="output--__invalidresult__"></a>Output: __non <Result> valido__[]

Matrice di risultati della misurazione.
