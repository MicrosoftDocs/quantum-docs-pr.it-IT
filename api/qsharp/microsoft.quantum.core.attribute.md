# <a name="attribute-user-defined-type"></a>Tipo definito dall'utente attribute

Spazio dei nomi: [Microsoft. Quantum. Core](xref:Microsoft.Quantum.Core)

Pacchetto: [Microsoft. Quantum. QSharp. Core](https://nuget.org/packages/Microsoft.Quantum.QSharp.Core)


Attributo riconosciuto dal compilatore utilizzato per contrassegnare i tipi definiti dall'utente come attributi.

```qsharp

newtype Attribute = (Unit);
```
