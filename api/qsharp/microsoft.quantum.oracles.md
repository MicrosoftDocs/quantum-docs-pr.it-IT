# <a name="microsoftquantumoracles-namespace"></a>Spazio dei nomi Microsoft. Quantum. Oracles

Questo spazio dei nomi fornisce tipi definiti dall'utente per decorare vari tipi di Oracle in base all'uso previsto negli algoritmi quantistici, ad esempio la stima della fase e l'amplificazione dell'ampiezza.
