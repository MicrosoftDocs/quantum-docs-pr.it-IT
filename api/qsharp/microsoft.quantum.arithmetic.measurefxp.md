# <a name="measurefxp-operation"></a>Operazione MeasureFxP

Spazio dei nomi: [Microsoft. Quantum. aritmetico](xref:Microsoft.Quantum.Arithmetic)

Pacchetto: [Microsoft. Quantum. Numerics](https://nuget.org/packages/Microsoft.Quantum.Numerics)


Misurare un numero a virgola fissa, restituisce il relativo valore come Double e Reimposta tutto il registro su zero.

```qsharp
operation MeasureFxP (fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Double
```


## <a name="input"></a>Input

### <a name="fp--fixedpoint"></a>FP: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Numero a virgola fissa da misurare.



## <a name="output--double"></a>Output: [Double](xref:microsoft.quantum.lang-ref.double)
