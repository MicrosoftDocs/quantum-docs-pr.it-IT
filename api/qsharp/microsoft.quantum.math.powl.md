# <a name="powl-function"></a>PowL (funzione)

Spazio dei nomi: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Pacchetto: [Microsoft. Quantum. standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Restituisce un numero elevato a una determinata potenza.

```qsharp
function PowL (a : BigInt, power : Int) : BigInt
```


## <a name="input"></a>Input

### <a name="a--bigint"></a>r: [bigint](xref:microsoft.quantum.lang-ref.bigint)

Numero $a $ da generare.


### <a name="power--int"></a>Potenza: [int](xref:microsoft.quantum.lang-ref.int)

Power $b $ a cui deve essere generato $a $.



## <a name="output--bigint"></a>Output: [bigint](xref:microsoft.quantum.lang-ref.bigint)

Power $a ^ b $
