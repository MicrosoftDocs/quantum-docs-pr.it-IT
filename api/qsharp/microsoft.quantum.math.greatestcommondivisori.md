# <a name="greatestcommondivisori-function"></a>GreatestCommonDivisorI (funzione)

Spazio dei nomi: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Pacchetto [](https://nuget.org/packages/)


Calcola il massimo comun divisore di $a $ e $b $. Il MCD è sempre positivo.

```qsharp
function GreatestCommonDivisorI (a : Int, b : Int) : Int
```


## <a name="input"></a>Input

### <a name="a--int"></a>a: [int](xref:microsoft.quantum.lang-ref.int)

il primo numero di cui è in corso il calcolo del divisore più comune esteso


### <a name="b--int"></a>b: [int](xref:microsoft.quantum.lang-ref.int)

secondo numero di cui è in corso il calcolo del divisore maggiore comune esteso



## <a name="output--int"></a>Output: [int](xref:microsoft.quantum.lang-ref.int)

Massimo comune divisore di $a $ e $b $
