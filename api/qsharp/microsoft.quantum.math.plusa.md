# <a name="plusa-function"></a>Funzione PLUSa

Spazio dei nomi: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Pacchetto: [Microsoft. Quantum. standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Restituisce la somma (concatenazione) di due input.

```qsharp
function PlusA<'Element> (a : 'Element[], b : 'Element[]) : 'Element[]
```


## <a name="input"></a>Input

### <a name="a--element"></a>a:' elemento []

Primo input $a $ da sommare.


### <a name="b--element"></a>b:' elemento []

Secondo input $b $ da sommare.



## <a name="output--element"></a>Output:' elemento []

La somma $a + b $.

## <a name="type-parameters"></a>Parametri di tipo

### <a name="element"></a>' Elemento

Tipo di ogni elemento in ognuna delle due matrici di input.
