# <a name="dirac-notation"></a><span data-ttu-id="a0363-182">Notazione Dirac</span><span class="sxs-lookup"><span data-stu-id="a0363-182">Dirac Notation</span></span>

<span data-ttu-id="a0363-183">Sebbene la notazione del vettore di colonna sia onnipresente nell'algebra lineare, è spesso complessa nell'elaborazione quantistica, soprattutto quando si gestiscono più qubits.</span><span class="sxs-lookup"><span data-stu-id="a0363-183">While column vector notation is ubiquitous in linear algebra, it is often cumbersome in quantum computing especially when dealing with multiple qubits.</span></span>  <span data-ttu-id="a0363-184">Ad esempio, quando si definisce come $ \psi $ vettore, non è chiaro se $ \psi $ è una riga o un vettore di colonna.</span><span class="sxs-lookup"><span data-stu-id="a0363-184">For example, when we define $\psi$ to be a vector it is not explicitly clear whether $\psi$ is a row or a column vector.</span></span>  <span data-ttu-id="a0363-185">Pertanto, se $ \phi $ e $ \psi $ sono vettori, non è altrettanto chiaro se $ \phi \psi $ è anche definito perché le forme di $ \phi $ e $ \psi $ potrebbero non essere chiare nel contesto.</span><span class="sxs-lookup"><span data-stu-id="a0363-185">Thus if $\phi$ and $\psi$ are vectors then it is equally unclear if $\phi \psi$ is even defined because the shapes of $\phi$ and $\psi$ may be unclear in the context.</span></span>  <span data-ttu-id="a0363-186">Oltre l'ambiguità sulle forme dei vettori, esprimere anche vettori semplici con la notazione algebrica lineare introdotta in precedenza può essere molto complessa.</span><span class="sxs-lookup"><span data-stu-id="a0363-186">Beyond the ambiguity about the shapes of vectors, expressing even simple vectors using the linear algebraic notation introduced earlier can be very cumbersome.</span></span> <span data-ttu-id="a0363-187">Se, ad esempio, si vuole descrivere uno $ $ stato n-qubit in cui ogni qubit accetta il valore $ 0, lo stato verrà $ espresso formalmente come</span><span class="sxs-lookup"><span data-stu-id="a0363-187">For example, if we wish to describe an $n$-qubit state where each qubit takes the value $0$ then we would formally express the state as</span></span> 

<span data-ttu-id="a0363-188">$$\begin{bmatrix}1 \\\\ 0 \end{bmatrix} \otimes \cdots \otimes \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} .</span><span class="sxs-lookup"><span data-stu-id="a0363-188">$$\begin{bmatrix}1 \\\\  0 \end{bmatrix}\otimes \cdots \otimes\begin{bmatrix}1 \\\\  0 \end{bmatrix}.</span></span> $$  

<span data-ttu-id="a0363-189">Ovviamente la valutazione di questo prodotto tensore è poco pratica perché il vettore si trova in uno spazio esponenzialmente grande e pertanto questa notazione è in realtà la migliore descrizione dello stato che è possibile assegnare utilizzando la notazione precedente.</span><span class="sxs-lookup"><span data-stu-id="a0363-189">Of course evaluating this tensor product is impractical because the vector lies in an exponentially large space and so this notation is in fact the best description of the state that can be given using the previous notation.</span></span>  

<span data-ttu-id="a0363-190">La [*notazione Dirac*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) risolve questi problemi presentando un nuovo linguaggio per soddisfare le esigenze specifiche dei meccanismi quantistici.</span><span class="sxs-lookup"><span data-stu-id="a0363-190">[*Dirac notation*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) solves these issues by presenting a new language to fit the precise needs of quantum mechanics.</span></span>  <span data-ttu-id="a0363-191">Per questo motivo, è consigliabile che il lettore non possa visualizzare gli esempi in questa sezione come una ricetta rigida su come descrivere gli Stati dei quantum, ma incoraggiare il lettore a visualizzarli come suggerimenti che possono essere usati per esprimere concisamente le idee di Quantum.</span><span class="sxs-lookup"><span data-stu-id="a0363-191">For this reason, we recommend the reader not view the examples in this section as a rigid prescription of how to describe quantum states, but rather encourage the reader to view these as suggestions that can be used to concisely express quantum ideas.</span></span>

<span data-ttu-id="a0363-192">Sono disponibili due tipi di vettori nella notazione Dirac: il vettore *Bra* e il vettore *KET* , denominati perché, quando assemblati, formano un *freno* o un prodotto interno.</span><span class="sxs-lookup"><span data-stu-id="a0363-192">There are two types of vectors in Dirac notation: the *bra* vector and the *ket* vector, so named because when put together they form a *braket* or inner product.</span></span>  <span data-ttu-id="a0363-193">Se $ \psi $ è un vettore di colonna, è possibile scriverlo nella notazione Dirac come $ \ket { \psi } $ , dove $ \ket { \cdot indica } $ che si tratta di un vettore di colonna di unità, ovvero un vettore *KET* .</span><span class="sxs-lookup"><span data-stu-id="a0363-193">If $\psi$ is a column vector then we can write it in Dirac notation as $\ket{\psi}$, where the $\ket{\cdot}$ denotes that it is a unit column vector, i.e., a *ket* vector.</span></span>  <span data-ttu-id="a0363-194">Analogamente, il vettore $ \psi ^ \dagger $ di riga viene espresso come $ \bra { \psi } $ .</span><span class="sxs-lookup"><span data-stu-id="a0363-194">Similarly, the row vector $\psi^\dagger$ is expressed as $\bra{\psi}$.</span></span> <span data-ttu-id="a0363-195">In altre parole, $ \psi ^ \dagger $ viene ottenuto applicando una coniugazione complessa per la voce per gli elementi della trasposta di $ \psi $ .</span><span class="sxs-lookup"><span data-stu-id="a0363-195">In other words, $\psi^\dagger$ is obtained by applying entry-wise complex conjugation to the elements of the transpose of $\psi$.</span></span> <span data-ttu-id="a0363-196">La notazione bra-ket implica direttamente che $ \braket { \psi | \psi } $ è il prodotto interno del vettore $ \psi $ con se stesso, che è per definizione $ 1 $ .</span><span class="sxs-lookup"><span data-stu-id="a0363-196">The bra-ket notation directly implies that $\braket{\psi|\psi}$ is the inner product of vector $\psi$ with itself, which is by definition $1$.</span></span>  

<span data-ttu-id="a0363-197">Più in generale, se $ \psi $ e $ \phi $ sono vettori di stato quantistici, il prodotto interno è $ \braket { \phi | \psi } $ il che implica che la probabilità di misurare lo stato $ \ket { \psi } $ $ \ket { \phi } $ è $ | \braket { \phi | \psi } | ^ 2 $ .</span><span class="sxs-lookup"><span data-stu-id="a0363-197">More generally, if $\psi$ and $\phi$ are quantum state vectors their inner product is $\braket{\phi|\psi}$ which implies that the probability of measuring the state $\ket{\psi}$ to be $\ket{\phi}$ is $|\braket{\phi|\psi}|^2$.</span></span>  

<span data-ttu-id="a0363-198">La convenzione seguente viene usata per descrivere gli Stati del quantum che codificano i valori di zero e uno (gli Stati di base di calcolo a qubit singolo):</span><span class="sxs-lookup"><span data-stu-id="a0363-198">The following convention is used to describe the quantum states that encode the values of zero and one (the single-qubit computational basis states):</span></span>

$$
<span data-ttu-id="a0363-199">\begin{bmatrix}1 \\\\ 0 \end{bmatrix} = \ket { 0 } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="a0363-199">\begin{bmatrix} 1 \\\\  0 \end{bmatrix} = \ket{0},\qquad</span></span>
<span data-ttu-id="a0363-200">\begin{bmatrix}0 \\\\ 1 \end{bmatrix} = \ket { 1 } .</span><span class="sxs-lookup"><span data-stu-id="a0363-200">\begin{bmatrix} 0 \\\\  1 \end{bmatrix} = \ket{1}.</span></span>
$$

### <a name="example-representing-the-hadamard-operation-with-dirac-notation"></a><span data-ttu-id="a0363-201">Esempio: rappresentazione dell'operazione Hadamard con la notazione Dirac</span><span class="sxs-lookup"><span data-stu-id="a0363-201">Example: representing the Hadamard operation with Dirac notation</span></span>

<span data-ttu-id="a0363-202">La notazione seguente viene spesso usata per descrivere gli Stati che risultano dall'applicazione di Hadamard Gate a $ \ket { 0 } $ e $ \ket { 1 } $ (che corrispondono ai vettori di unità nelle $ direzioni + x $ e $ -x $ sulla sfera Bloch):</span><span class="sxs-lookup"><span data-stu-id="a0363-202">The following notation is often used to describe the states that result from applying the Hadamard gate to $\ket{0}$ and $\ket{1}$ (which correspond to the unit vectors in the $+x$ and $-x$ directions on the Bloch sphere):</span></span>

$$
<span data-ttu-id="a0363-203">\frac{1 } { \sqrt { 2 } } \begin{bmatrix} 1 \\\\ \end{bmatrix} = H \ket { 0 } = \ket { + } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="a0363-203">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  1 \end{bmatrix}=H\ket{0} = \ket{+},\qquad</span></span>
<span data-ttu-id="a0363-204">\frac{1 } { \sqrt { 2 } } \begin{bmatrix} 1 \\\\ -1 \end{bmatrix} = H \ket { 1 } = \ket { - } .</span><span class="sxs-lookup"><span data-stu-id="a0363-204">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  -1 \end{bmatrix} =H\ket{1} = \ket{-} .</span></span>
$$

<span data-ttu-id="a0363-205">Questi Stati possono anche essere espansi usando la notazione Dirac come somme di $ \ket { 0 } $ e $ \ket { 1 } $ :</span><span class="sxs-lookup"><span data-stu-id="a0363-205">These states can also be expanded using Dirac notation as sums of $\ket{0}$ and $\ket{1}$:</span></span>

$$
<span data-ttu-id="a0363-206">\ket{+}= \frac{ 1 } { \sqrt { 2 } } ( \ket { 0 }  +  \ket { 1 } ), \qquad \ket { - } = \frac { 1 } { \sqrt { 2 } } ( \ket { 0 }  -  \ket { 1 } ).</span><span class="sxs-lookup"><span data-stu-id="a0363-206">\ket{+} = \frac{1}{\sqrt{2}}(\ket{0} + \ket{1}),\qquad \ket{-} = \frac{1}{\sqrt{2}}(\ket{0} - \ket{1}).</span></span>
$$

### <a name="computational-basis-vectors"></a><span data-ttu-id="a0363-207">Vettori di base di calcolo</span><span class="sxs-lookup"><span data-stu-id="a0363-207">Computational basis vectors</span></span>
