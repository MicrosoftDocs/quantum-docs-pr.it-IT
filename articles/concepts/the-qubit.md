# <a name="the-qubit"></a><span data-ttu-id="b42f3-182">Qubit</span><span class="sxs-lookup"><span data-stu-id="b42f3-182">The Qubit</span></span>

<span data-ttu-id="b42f3-183">Proprio come bits sono l'oggetto fondamentale delle informazioni nell'elaborazione classica, [*qubits*](https://en.wikipedia.org/wiki/Qubit) (Quantum BITS) sono l'oggetto fondamentale delle informazioni in quantum computing.</span><span class="sxs-lookup"><span data-stu-id="b42f3-183">Just as bits are the fundamental object of information in classical computing, [*qubits*](https://en.wikipedia.org/wiki/Qubit) (quantum bits) are the fundamental object of information in quantum computing.</span></span>  <span data-ttu-id="b42f3-184">Per comprendere questa corrispondenza, esaminiamo l'esempio più semplice: un singolo qubit.</span><span class="sxs-lookup"><span data-stu-id="b42f3-184">To understand this correspondence, let's look at the simplest example: a single qubit.</span></span>

## <a name="representing-a-qubit"></a><span data-ttu-id="b42f3-185">Rappresentazione di un qubit</span><span class="sxs-lookup"><span data-stu-id="b42f3-185">Representing a Qubit</span></span>

<span data-ttu-id="b42f3-186">Mentre un bit, o una cifra binaria, può avere valore $ 0 $ o $ 1 $ , un qubit può avere un valore che è uno di questi o una superposizione quantistica pari a $ 0 $ e $ 1 $ .</span><span class="sxs-lookup"><span data-stu-id="b42f3-186">While a bit, or binary digit, can have value either $0$ or $1$, a qubit can have a value that is either of these or a quantum superposition of $0$ and $1$.</span></span>

<span data-ttu-id="b42f3-187">Lo stato di un singolo qubit può essere descritto da un vettore di colonna bidimensionale di unità Norm, ovvero la grandezza quadrata delle relative voci deve essere sommata a $ 1 $ .</span><span class="sxs-lookup"><span data-stu-id="b42f3-187">The state of a single qubit can be described by a two-dimensional column vector of unit norm, that is, the magnitude squared of its entries must sum to $1$.</span></span> <span data-ttu-id="b42f3-188">Questo vettore, denominato quantum state Vector, include tutte le informazioni necessarie per descrivere il sistema Quantum One-qubit, proprio come un singolo bit, che include tutte le informazioni necessarie per descrivere lo stato di una variabile binaria.</span><span class="sxs-lookup"><span data-stu-id="b42f3-188">This vector, called the quantum state vector, holds all the information needed to describe the one-qubit quantum system just as a single bit holds all of the information needed to describe the state of a binary variable.</span></span>

<span data-ttu-id="b42f3-189">Qualsiasi vettore di colonna bidimensionale di numeri reali o complessi con la norma $ 1 $ rappresenta un possibile stato quantum utilizzato da un qubit.</span><span class="sxs-lookup"><span data-stu-id="b42f3-189">Any two-dimensional column vector of real or complex numbers with norm $1$ represents a possible quantum state held by a qubit.</span></span> <span data-ttu-id="b42f3-190">$ \begin{bmatrix} \alpha \\\\ \beta \end{bmatrix} $ Rappresenta pertanto uno stato qubit se $ \alpha $ e $ \beta $ sono numeri complessi che soddisfano $ | \alpha | ^ 2 + | \beta | ^ 2 = 1 $ .  </span><span class="sxs-lookup"><span data-stu-id="b42f3-190">Thus $\begin{bmatrix} \alpha \\\\  \beta \end{bmatrix}$ represents a qubit state if $\alpha$ and $\beta$ are complex numbers satisfying $|\alpha|^2 + |\beta|^2 = 1$.</span></span> <span data-ttu-id="b42f3-191">Alcuni esempi di vettori di stato quantum validi che rappresentano qubits includono</span><span class="sxs-lookup"><span data-stu-id="b42f3-191">Some examples of valid quantum state vectors representing qubits include</span></span>

<span data-ttu-id="b42f3-192">$$\begin{bmatrix}1 \\\\ 0 \end{bmatrix} , \begin{bmatrix} 0 \\\\ 1 \end{bmatrix} , \begin{bmatrix} \frac { 1 2 } { \sqrt { } } \\\\ \frac { 1 } { \sqrt { 2 } } \end{bmatrix} , \begin{bmatrix} \frac { 1 } { \sqrt { 2 } } \\\\ \frac { -1 } { \sqrt { 2 } } \end{bmatrix} \text { e } \begin{bmatrix} \frac { 1 } { \sqrt { 2 } } \\\\ \frac { i } { \sqrt { 2 } } \end{bmatrix} .      $$</span><span class="sxs-lookup"><span data-stu-id="b42f3-192">$$\begin{bmatrix} 1 \\\\  0 \end{bmatrix}, \begin{bmatrix} 0 \\\\  1 \end{bmatrix}, \begin{bmatrix} \frac{1}{\sqrt{2}} \\\\  \frac{1}{\sqrt{2}} \end{bmatrix}, \begin{bmatrix} \frac{1}{\sqrt{2}} \\\\  \frac{-1}{\sqrt{2}} \end{bmatrix}, \text{ and }\begin{bmatrix} \frac{1}{\sqrt{2}} \\\\  \frac{i}{\sqrt{2}} \end{bmatrix}.$$</span></span>

<span data-ttu-id="b42f3-193">I vettori di stato quantum $ \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} $ e $ \begin{bmatrix} 0 \\\\ 1 \end{bmatrix} $ accettano un ruolo speciale.</span><span class="sxs-lookup"><span data-stu-id="b42f3-193">The quantum state vectors $\begin{bmatrix} 1 \\\\  0 \end{bmatrix}$ and $\begin{bmatrix} 0 \\\\  1 \end{bmatrix}$ take a special role.</span></span> <span data-ttu-id="b42f3-194">Questi due vettori formano una base per lo spazio vettoriale che descrive lo stato di qubit.</span><span class="sxs-lookup"><span data-stu-id="b42f3-194">These two vectors form a basis for the vector space that describes the qubit's state.</span></span> <span data-ttu-id="b42f3-195">Ciò significa che qualsiasi vettore di stato quantum può essere scritto come una somma di questi vettori di base.</span><span class="sxs-lookup"><span data-stu-id="b42f3-195">This means that any quantum state vector can be written as a sum of these basis vectors.</span></span> <span data-ttu-id="b42f3-196">In particolare, il vettore $ \begin{bmatrix} x \\\\ y \end{bmatrix} $ può essere scritto come $ x \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} + y \begin{bmatrix} 0 \\\\ 1 \end{bmatrix} $ .</span><span class="sxs-lookup"><span data-stu-id="b42f3-196">Specifically, the vector $\begin{bmatrix} x \\\\  y \end{bmatrix}$ can be written as $x \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} + y \begin{bmatrix} 0 \\\\  1 \end{bmatrix}$.</span></span> <span data-ttu-id="b42f3-197">Sebbene la rotazione di questi vettori possa essere usata come base perfettamente valida per qubit, si sceglie di privilegiare questo valore, chiamandolo come *base computazionale* .</span><span class="sxs-lookup"><span data-stu-id="b42f3-197">While any rotation of these vectors would serve as a perfectly valid basis for the qubit, we choose to privilege this one, by calling it the *computational basis* .</span></span>

<span data-ttu-id="b42f3-198">Questi due stati Quantum corrispondono ai due stati di un bit classico, ovvero $ 0 $ e $ 1 $ .</span><span class="sxs-lookup"><span data-stu-id="b42f3-198">We take these two quantum states to correspond to the two states of a classical bit, namely $0$ and $1$.</span></span> <span data-ttu-id="b42f3-199">La convenzione standard consiste nel scegliere</span><span class="sxs-lookup"><span data-stu-id="b42f3-199">The standard convention is to choose</span></span>

<span data-ttu-id="b42f3-200">$$0 \equiv \begin{bmatrix} 1 \\\\  0 \end{bmatrix} , \qquad 1 \equiv \begin{bmatrix} 0 \\\\  1 \end{bmatrix} ,$$</span><span class="sxs-lookup"><span data-stu-id="b42f3-200">$$0\equiv \begin{bmatrix} 1 \\\\  0 \end{bmatrix}, \qquad 1 \equiv \begin{bmatrix} 0 \\\\  1 \end{bmatrix},$$</span></span>

<span data-ttu-id="b42f3-201">Sebbene la scelta opposta possa essere eseguita ugualmente correttamente.</span><span class="sxs-lookup"><span data-stu-id="b42f3-201">although the opposite choice could equally well be taken.</span></span> <span data-ttu-id="b42f3-202">Pertanto, dal numero infinito di possibili vettori di stato quantum a singolo qubit, solo due corrispondono agli Stati dei bit classici; tutti gli altri Stati quantum non lo sono.</span><span class="sxs-lookup"><span data-stu-id="b42f3-202">Thus, out of the infinite number of possible single-qubit quantum state vectors, only two correspond to states of classical bits; all other quantum states do not.</span></span>

## <a name="measuring-a-qubit"></a><span data-ttu-id="b42f3-203">Misurazione di un qubit</span><span class="sxs-lookup"><span data-stu-id="b42f3-203">Measuring a Qubit</span></span>

<span data-ttu-id="b42f3-204">Ora che si è appreso come rappresentare una qubit, è possibile ottenere alcune informazioni sull'aspetto di questi stati esaminando il concetto di [*misurazione*](https://en.wikipedia.org/wiki/Measurement_in_quantum_mechanics).</span><span class="sxs-lookup"><span data-stu-id="b42f3-204">Now that we know how to represent a qubit, we can gain some intuition for what these states represent by discussing the concept of [*measurement*](https://en.wikipedia.org/wiki/Measurement_in_quantum_mechanics).</span></span> <span data-ttu-id="b42f3-205">Una misura corrisponde all'idea informale di "ricerca" in un qubit, che comprime immediatamente lo stato del quantum in uno dei due stati classici $ \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} $ o $ \begin{bmatrix} 0 \\\\ 1 \end{bmatrix} $ .</span><span class="sxs-lookup"><span data-stu-id="b42f3-205">A measurement corresponds to the informal idea of “looking” at a qubit, which immediately collapses the quantum state to one of the two classical states  $\begin{bmatrix} 1 \\\\  0 \end{bmatrix}$ or  $\begin{bmatrix} 0 \\\\  1 \end{bmatrix}$.</span></span> <span data-ttu-id="b42f3-206">Quando viene misurato un qubit fornito dal vettore di stato quantum $ \begin{bmatrix} \alpha \\\\ \beta \end{bmatrix} $ , viene ottenuto il risultato $ 0 $ con probabilità $ | \alpha | ^ 2 $ e il risultato $ 1 $ con probabilità $ | \beta | ^ 2 $ .  </span><span class="sxs-lookup"><span data-stu-id="b42f3-206">When a qubit given by the quantum state vector  $\begin{bmatrix} \alpha \\\\  \beta \end{bmatrix}$ is measured, we obtain the outcome $0$ with probability $|\alpha|^2$ and the outcome $1$  with probability $|\beta|^2$.</span></span> <span data-ttu-id="b42f3-207">Sul risultato $ 0 $ , il nuovo stato di qubit è pari a $ \begin{bmatrix} 1 \\\\ 0 \end{bmatrix} $ ;
