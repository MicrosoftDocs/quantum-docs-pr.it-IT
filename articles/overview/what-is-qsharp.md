## <a name="how-does-q-work"></a><span data-ttu-id="d05a8-111">Come funziona il linguaggio Q#?</span><span class="sxs-lookup"><span data-stu-id="d05a8-111">How does Q# work?</span></span>

<span data-ttu-id="d05a8-112">Uno degli pilastri del linguaggio Q# è il tipo `Qubit`, che non prevede la copia o l'accesso diretto, proprio come un qubit.</span><span class="sxs-lookup"><span data-stu-id="d05a8-112">One of the fundamental building blocks of Q# is the `Qubit` type, which cannot be copied or directly accessed, just like a real qubit.</span></span> <span data-ttu-id="d05a8-113">È invece possibile misurarlo e archiviare il risultato della misurazione in una variabile `Result`, un tipo Q# che accetta due valori possibili, ovvero `Zero` e `One`.</span><span class="sxs-lookup"><span data-stu-id="d05a8-113">Instead, we can measure it and store the outcome of the measurement in a `Result` variable, a Q# type that can take two possible values: `Zero` and `One`.</span></span> <span data-ttu-id="d05a8-114">Costrutti come questo garantiscono che gli algoritmi rispettino sempre le leggi della fisica quantistica e possono essere eseguiti correttamente in computer o simulatori quantistici.</span><span class="sxs-lookup"><span data-stu-id="d05a8-114">Constructs like this one guarantee that algorithms always respect the laws of quantum physics and can run correctly on quantum computers or simulators.</span></span>

<span data-ttu-id="d05a8-115">Q# include anche funzionalità di logica classica, come cicli o istruzioni condizionali con alcune particolarità, per garantire il rispetto di tutte le regole quantistiche.</span><span class="sxs-lookup"><span data-stu-id="d05a8-115">Q# also includes classical logic features like conditionals or loops with some subtleties to make sure that all the quantum rules are being respected.</span></span> <span data-ttu-id="d05a8-116">Le operazioni quantistiche devono, ad esempio, essere reversibili.</span><span class="sxs-lookup"><span data-stu-id="d05a8-116">For example, quantum operations need to be reversible.</span></span> <span data-ttu-id="d05a8-117">Questa regola impone alcuni vincoli sulla modalità di esecuzione dei cicli.</span><span class="sxs-lookup"><span data-stu-id="d05a8-117">This enforces some constraints on the way loops are executed.</span></span>

<span data-ttu-id="d05a8-118">I programmi Q# sono spesso abbinati a un programma host scritto in C# o Python, che costituisce una soluzione pratica per organizzare codice classico e codice quantistico.</span><span class="sxs-lookup"><span data-stu-id="d05a8-118">Q# programs are often paired with a host program written in C# or Python, which can provide convenient organization of classical and quantum code.</span></span> <span data-ttu-id="d05a8-119">Oltre a supportare linguaggi .NET, come C# e Python, il QDK include il supporto per Jupyter Notebook con il kernel Jupyter IQ#.</span><span class="sxs-lookup"><span data-stu-id="d05a8-119">In addition to supporting .NET languages such as C# and Python, the QDK provides Jupyter Notebook support with the IQ# Jupyter kernel.</span></span>
