<span data-ttu-id="ec320-106">Il QDK è costituito da un'ampia gamma di strumenti che offrono agli sviluppatori tutto ciò che serve per iniziare a scrivere programmi quantistici.</span><span class="sxs-lookup"><span data-stu-id="ec320-106">The QDK comprises a wide range of tools which give developers everything they need to start writing quantum programs.</span></span>

## <a name="write-quantum-programs-not-quantum-circuits"></a><span data-ttu-id="ec320-121">Scrivere programmi quantistici, non circuiti quantistici</span><span class="sxs-lookup"><span data-stu-id="ec320-121">Write quantum programs, not quantum circuits</span></span>

<span data-ttu-id="ec320-123">Anche se il modello a circuiti è stato utile per molti anni nella ricerca del calcolo quantistico, in Microsoft si ritiene che gli sviluppatori debbano superare i circuiti quantistici e sviluppare applicazioni e algoritmi quantistici usando Q#.</span><span class="sxs-lookup"><span data-stu-id="ec320-123">While the circuit model has been useful for many years in quantum computing research, here at Microsoft, we believe that developers can go beyond quantum circuits and develop quantum algorithms and applications using Q#.</span></span>
<span data-ttu-id="ec320-124">Il linguaggio Q# è il frutto dell'esperienza maturata in decenni di sviluppo di software classico e offre agli sviluppatori quantistici funzionalità del linguaggio generali destinate al calcolo quantistico.</span><span class="sxs-lookup"><span data-stu-id="ec320-124">The Q# language was built to take advantage of what we’ve learned through decades of classical software development, and empower quantum developers with high-level language functionality targeted for quantum computing.</span></span>

## <a name="how-does-q-work"></a><span data-ttu-id="ec320-125">Come funziona il linguaggio Q#?</span><span class="sxs-lookup"><span data-stu-id="ec320-125">How does Q# work?</span></span>

<span data-ttu-id="ec320-126">Il linguaggio di programmazione Q# offre un set intuitivo di tipi, operazioni ed espressioni logiche per sviluppare algoritmi senza doversi preoccupare della logica interna del computer quantistico.</span><span class="sxs-lookup"><span data-stu-id="ec320-126">The Q# programming language provides you with an intuitive set of types, operations, and logic expressions to develop algorithms without having to worry about the internal logic of the quantum computer.</span></span>

<span data-ttu-id="ec320-127">Uno degli pilastri del linguaggio Q# è il tipo `Qubit`, che non prevede la copia o l'accesso diretto, proprio come un qubit.</span><span class="sxs-lookup"><span data-stu-id="ec320-127">One of the fundamental building blocks of Q# is the `Qubit` type, which cannot be copied or directly accessed, just like a real qubit.</span></span>
<span data-ttu-id="ec320-128">È invece possibile misurarlo e archiviare il risultato della misurazione in una variabile `Result`, un tipo Q# che accetta due valori possibili, ovvero `Zero` e `One`.</span><span class="sxs-lookup"><span data-stu-id="ec320-128">Instead, we can measure it and store the outcome of the measurement in a `Result` variable, a Q# type that can take two possible values: `Zero` and `One`.</span></span>
<span data-ttu-id="ec320-129">Costrutti come questo garantiscono che gli algoritmi rispettino sempre le leggi della fisica quantistica e possono essere eseguiti correttamente in computer o simulatori quantistici.</span><span class="sxs-lookup"><span data-stu-id="ec320-129">Constructs like this one guarantee that algorithms always respect the laws of quantum physics and can run correctly on quantum computers or simulators.</span></span>

<span data-ttu-id="ec320-130">Q# include anche funzionalità di logica classica, come cicli e istruzioni condizionali con alcune particolarità, per garantire il rispetto di tutte le regole quantistiche.</span><span class="sxs-lookup"><span data-stu-id="ec320-130">Q# also includes classical logic features like conditionals and loops with some subtleties to make sure that all the quantum rules are being respected.</span></span>
<span data-ttu-id="ec320-131">Ad esempio, è possibile vincolare la modalità di esecuzione dei cicli per garantire che le operazioni quantistiche non vengano chiamate all'interno di funzioni che possono contenere solo subroutine classiche deterministiche.</span><span class="sxs-lookup"><span data-stu-id="ec320-131">For example, constraining the way loops are executed to ensure that quantum operations are not called within functions which may only contain deterministic classical subroutines.</span></span>

<span data-ttu-id="ec320-132">I programmi Q# sono spesso abbinati a un programma host scritto in C# o Python, che costituisce una soluzione pratica per organizzare codice classico e codice quantistico.</span><span class="sxs-lookup"><span data-stu-id="ec320-132">Q# programs are often paired with a host program written in C# or Python, which can provide convenient organization of classical and quantum code.</span></span>
<span data-ttu-id="ec320-133">Oltre a supportare linguaggi come C# e Python, il QDK include il supporto per Jupyter Notebook con il kernel Jupyter IQ#.</span><span class="sxs-lookup"><span data-stu-id="ec320-133">In addition to supporting languages such as C# and Python, the QDK provides Jupyter Notebook support with the IQ# Jupyter kernel.</span></span>
